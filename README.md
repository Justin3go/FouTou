## 简介
这是一份基于区块链的摄影社区系统，你可以简单理解为简化版的opensea，但是其中的nft并不是遵守的erc721标准，而是将其中的所有权交易改为了使用权交易，创作者打造的nft只归自己所有，整个系统几乎都是围绕着这一点出发的
